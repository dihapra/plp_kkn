Nama-nama Kolom Tabel principal

id → Primary Key (int, auto increment)

school_id → ID sekolah (int, foreign key ke tabel school kemungkinan)

nama → Nama kepala sekolah (varchar)

email → Alamat email (varchar)

phone → Nomor telepon (varchar)

bank → Nama bank (varchar)

account_number → Nomor rekening (varchar)

account_name → Nama pemilik rekening (varchar)

book → Nomor/seri buku tabungan (varchar)

nik → Nomor Induk Kependudukan (varchar)

identification_card → Dokumen identitas (misalnya KTP, file path atau nomor) (varchar)

status → Status menikah nntinya

status_data → Status data administratif (varchar)

created_at → Timestamp dibuat

updated_at → Timestamp diperbarui

updated_by → ID user yang terakhir memperbarui

verified_by → ID user yang memverifikasi
